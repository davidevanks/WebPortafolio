whatsApp Bot

Proyecto en .Net 5 Framework

Este proyecto es un Bot en whatsApp usando el servicio de microsoft QnA Maker, en este caso use una base de conocimiento basada en el blog de Plata con plática

Para usarlo, agrega a tu agenda el siguiente número de contacto: +14155238886 e inicia el bot mandando el mensaje: join swam-rear o da click en el siguiente enlace

Puedes preguntar preguntas como: En que invertir en Nicaragua?, Como ahorrar? o cualquier otra pregunta básica de finanzas personales.

Puedes ver el código fuente en mi GitHub

Utilice .NET 5 Framework, QnA Maker Microsoft Service, lenguaje backend C# y Twilio API.
